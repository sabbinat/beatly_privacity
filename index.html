Política de Privacidad – Beatly

Última actualización: 8 de julio de 2025

Beatly es una aplicación educativa que permite escuchar y descargar canciones a partir de un archivo JSON proporcionado por el docente. Respetamos tu privacidad y no recopilamos ningún tipo de información personal ni sensible.

1. Recolección de datos

La aplicación no recopila, almacena, ni comparte datos personales del usuario. Tampoco se requiere registro ni inicio de sesión.

2. Fuente del contenido

Las canciones disponibles en la aplicación provienen exclusivamente de un archivo JSON proporcionado por el profesor del curso. Beatly no accede a servicios de streaming ni a bibliotecas musicales públicas o privadas en línea.

3. Permisos solicitados

Beatly solicita ciertos permisos técnicos necesarios para su funcionamiento:

  • Acceso a almacenamiento: para guardar canciones descargadas localmente.
  • Acceso a red: para obtener el archivo JSON desde la URL proporcionada por el docente.
  • Ejecutar en segundo plano: para completar las descargas mientras la app no está activa.

Estos permisos no se utilizan para recopilar datos del usuario ni acceder a sus archivos personales.

4. Servicios de terceros

Beatly no utiliza servicios de terceros, bibliotecas analíticas ni SDKs que recolecten información personal o realicen seguimiento del usuario.

5. Público objetivo

La aplicación no está diseñada específicamente para niños menores de 13 años, pero puede ser utilizada por ellos bajo supervisión educativa. No incluye contenido inapropiado.

6. Cambios en esta política

Cualquier cambio relevante en esta política será comunicado dentro de la aplicación o a través del canal educativo correspondiente.
