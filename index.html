Política de Privacidad – Beatly

Última actualización: 8 de julio de 2025

Beatly es una aplicación que permite escuchar y descargar canciones a partir de un archivo JSON como parte de una actividad académica. Respetamos tu privacidad y no recopilamos ningún tipo de información personal.

1. Recolección de Datos

La aplicación no recopila, almacena, ni comparte datos personales del usuario.

2. Fuente del contenido

Las canciones disponibles en la aplicación provienen de un archivo JSON proporcionado por el docente responsable del curso. Beatly no accede a servicios externos ni a bibliotecas de música públicas o privadas en internet.

3. Permisos solicitados

Beatly solicita los siguientes permisos únicamente para su funcionamiento técnico:

  • Acceso a almacenamiento: para guardar canciones descargadas localmente.
  • Acceso a red: para obtener el archivo JSON con la lista de canciones desde la URL proporcionada por el profesor.
  • Ejecutar en segundo plano: para realizar descargas con notificaciones mientras la app no está en primer plano.

Estos permisos no implican recolección de datos ni acceso a archivos del usuario.

4. Servicios de terceros

Beatly no utiliza servicios de terceros ni bibliotecas que recolecten información personal o realicen seguimiento del usuario.

5. Cambios en esta política

Cualquier actualización de esta política será comunicada dentro de la aplicación o mediante el canal educativo correspondiente.
